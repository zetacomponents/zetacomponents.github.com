ezcQueryInvalidException __construct( string $type , string $message )

Constructs a QueryInvalid exception with the type $type and the additional information $message.

$type should be used to specify the type of the query that failed. Possible values are SELECT, INSERT, UPDATE and DELETE.

Use $message to specify exactly what went wrong.
